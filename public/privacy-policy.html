Política de Privacidade - Triia

Última atualização: Janeiro 2025

1. Informações que Coletamos

O Triia coleta as seguintes informações para fornecer serviços de triagem médica:

  • Dados de Saúde: Sintomas, sinais vitais, histórico médico
  • Dados Biométricos: Análise facial para rPPG, gravações de voz
  • Dados do Dispositivo: Informações técnicas para funcionamento do app
  • Dados de Uso: Interações com o aplicativo para melhorias

2. Como Usamos suas Informações

  • Realizar triagem médica e análise de sintomas
  • Gerar relatórios e recomendações de saúde
  • Melhorar a precisão dos algoritmos de IA
  • Fornecer suporte técnico e atendimento

3. Proteção de Dados

Implementamos medidas de segurança rigorosas:

  • Criptografia end-to-end para dados sensíveis
  • Armazenamento seguro com Supabase
  • Acesso restrito apenas a pessoal autorizado
  • Auditorias regulares de segurança

4. Compartilhamento de Dados

Seus dados médicos NUNCA são compartilhados com terceiros sem seu consentimento explícito, exceto:

  • Quando exigido por lei
  • Em situações de emergência médica
  • Com provedores de serviços sob acordos de confidencialidade

5. Seus Direitos

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir informações incorretas
  • Solicitar exclusão de dados
  • Portabilidade de dados
  • Retirar consentimento a qualquer momento

6. Retenção de Dados

Mantemos seus dados pelo tempo necessário para:

  • Fornecer serviços contínuos
  • Cumprir obrigações legais
  • Resolver disputas

7. Conformidade Legal

O Triia está em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD)
  • Regulamentações de dispositivos médicos
  • Padrões internacionais de privacidade
